* Get Controller's route template from constant value

* Swagger
* Logging
* Token Authentication
* WebSocket
* Reload endpoint
* Docker
* Long running jobs
* Simulate delay and errors
* Upsert option for PUT
* Eager data reload option
* Id field can be integer or string
